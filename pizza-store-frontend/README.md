# 🍕 Pizzeria Frontend

This is the React-based frontend for the Pizzeria application.

## 🚀 Quick Start Guide

Follow these steps to get the frontend running:

1.  **Open your terminal** and navigate to this folder:
    ```bash
    cd pizza-store-frontend
    ```
2.  **Install the necessary packages**:
    ```bash
    npm install
    ```
3.  **Start the development server**:
    ```bash
    npm start
    ```
4.  **Open your browser**: Go to `http://localhost:3000` to see the app!

## 🛠️ Features
- **Redux State Management**: Centralized store for auth, cart, and orders.
- **Premium UI**: Custom CSS with glassmorphism and smooth animations.
- **Responsive Design**: Optimized for Desktop and Mobile (iPhone 12).
